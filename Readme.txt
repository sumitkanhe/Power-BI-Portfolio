# Enterprise Financial Performance Dashboard (Power BI)

## Overview
This project is an executive-level financial analytics dashboard built using the Microsoft Power BI Financials sample dataset. It is designed to deliver insights into revenue performance, profitability, product trends, and geographic sales distribution.

The solution follows enterprise BI practices including star schema modeling, DAX-based KPI development, and executive-focused dashboard design.

---

## Business Objective
To provide a centralized financial analytics solution that enables stakeholders to:
- Monitor revenue and profit performance
- Analyze key business drivers
- Evaluate product and segment performance
- Assess regional sales distribution
- Track financial trends over time

---

## Dashboard Pages
- **Executive Overview** – High-level KPIs and performance trends  
- **Revenue & Profitability** – Margin analysis and financial breakdowns  
- **Product & Segment Analysis** – Business unit performance insights  
  

---

## Data Model
The project uses a star schema design:
- Fact Table: Financials
- Dimensions: Calendar, Product, Segment, Geography

---

## Key Metrics (DAX)
- Total Revenue  
- Gross Profit  
- Gross Margin %  
- Revenue Growth (YoY)  

---

## Design Approach
- Executive-level dashboard layout
- Consistent enterprise color theme
- Interactive slicers and drill-through
- Clean and minimal UI design
- Focus on storytelling and decision-making

---

## Tools Used
- Microsoft Power BI
- Power Query
- DAX

---

## Key Outcome
This dashboard enables fast, data-driven financial decision-making by providing a clear and structured view of business performance across multiple dimensions.
